1. ‚úÖ Are you connected to the office WiFi?
2. ‚úÖ Is the main office laptop turned on?
3. ‚úÖ Try refreshing the page
4. ‚úÖ Ask if web server is running

### Problem: Page loads slowly

**Solution:**
- Check WiFi signal strength
- Try a different browser (Chrome, Firefox, Edge)
- Close other browser tabs

### Problem: Can't see recent data

**Solution:**
- Refresh your browser page
- The system updates when you reload the page
- Ask if new data was just added to the desktop app

### Problem: "Connection refused"

**Solution:**
- The web server might have been stopped
- Contact the system administrator to restart it
- This happens when the main laptop is restarted

## Frequently Asked Questions

**Q: Can I add new documents?**
A: No, this is a read-only system. Use the desktop app to add/edit data.

**Q: Can I edit existing documents?**
A: No. Only viewing is allowed from the web browser.

**Q: What if I need to make changes?**
A: Contact the person who manages the desktop application.

**Q: Can I access this from home?**
A: Only if connected to the office network (VPN might be required).

**Q: What if the URL changes?**
A: The administrator will provide the new URL. Bookmark it.

**Q: Can multiple people use it at the same time?**
A: Yes! Unlimited users can access simultaneously.

**Q: Do I need an account or password?**
A: No, open access for anyone on the network.

**Q: Will my browsing be tracked?**
A: No, this is not an authentication system.

**Q: Can I download data?**
A: You can print and save as PDF from your browser.

**Q: How often is the data updated?**
A: Real-time - reload the page to see latest changes.
